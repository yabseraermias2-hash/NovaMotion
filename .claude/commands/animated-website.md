# Animated Website Build Rules

You are building a polished animated website from the project brief.

## Requirements

- Read the project brief first.
- Build the website using HTML, CSS, and JavaScript.
- Main file must be index.html.
- Use modern responsive design.
- Use smooth animations and transitions.
- Use images from the images folder when available.
- Keep the code clean and organized.
- Do not expose API keys or private information.
- Make sure the website works on desktop and mobile.

## Design Rules

- Use a cinematic modern style.
- Strong hero section.
- Clear typography.
- Smooth scrolling.
- Animated cards or sections.
- Good spacing and visual hierarchy.
- Professional color palette.
- Mobile-friendly layout.

## Final Check

Before finishing, make sure:
- index.html works.
- CSS and JavaScript are connected correctly.
- No broken image paths.
- Website is responsive.
- Animations are smooth.
- The final page matches the brief.
